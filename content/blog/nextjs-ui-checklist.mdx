Before I deliver a UI project, I quickly verify:

- No console errors
- Responsive layout (mobile/tablet/desktop)
- Basic accessibility (labels, focus, keyboard)
- Consistent spacing and typography
- SEO basics (title/description)

This reduces revisions and keeps delivery smooth.
